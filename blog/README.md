# Blog

Drop-in articles for a [Jekyll](https://jekyllrb.com/) site.

Each `.md` file here is a complete Jekyll post: a `YYYY-MM-DD-title.md` filename
plus YAML front matter (`layout`, `title`, `date`, `categories`, `tags`,
`excerpt`). To publish on a Jekyll site, copy the file into the site's `_posts/`
directory — no other changes needed. The body is standard
[kramdown](https://kramdown.gettalong.org/) Markdown (tables, fenced code, etc.).

Adjust `layout:` to match a layout your theme provides (most themes ship a
`post` layout).
