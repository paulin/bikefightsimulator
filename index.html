MentalPlayground

A collection of single-page JavaScript apps built for learning — visual, self-contained, and runnable with no build step. Each one makes an idea you can usually only read about something you can watch happen.

Bikefight RL Simulator

Two vehicles fight in an arena while a blue learner trains against a scripted bot using a TensorFlow.js deep Q-network. Place obstacles, watch the win rate climb, and open the live "Inside the Brain" panel to see the network's observations, neuron activations, and Q-values update in real time.

Data Science Playground

A visual sandbox for rebuilding intuition about machine-learning algorithms. Drag the data, turn the hyperparameter knobs, and watch the algorithm think in real time — 20 interactive screens spanning regression, classification, trees and ensembles, clustering, neural nets, CNNs, RNNs, attention, and reinforcement learning, each with its internals laid bare.
